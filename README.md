# Exploratory Data Analysis on Titanic Dataset

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify trends, patterns, and relationships affecting passenger survival.

## Objectives
- Clean and preprocess data
- Perform statistical analysis
- Visualize important patterns
- Identify key influencing factors

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Females had higher survival rates
- First-class passengers survived more
- Fare influenced survival probability
- Age distribution impacted survival trends
